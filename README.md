# Mini Photobooth!

A cute photobooth that takes pictures and prints them out for you!

<img width="566" height="423" alt="image" src="https://github.com/user-attachments/assets/921fa562-197a-44a0-a82e-ba4e782e4dec" />

I came up with this project because I thought it would be a good idea to move away from the audio based projects I was doing before and do something more physical. At first, I was thinking a digicam like thing in a tamogatchi syle case, but then I thought it would be cool to have a mini polaroid like photobooth, where people could come in room and take a quick photo! Obviously it won't be the most high quality, but it will be cute nonetheless! This will be a fun but practical accessory that I will definetly be using regularly as someone who is a fan of physical media! Some special things I did for this project include the fact that it is my first electronics project without a pcb, it has friction fit walls that snap togehter to make a case, and it includes a fidget buttons that doesnt do anything next to the shutter button (for funsies) along with a shutdown button.

Case Notes:
- (right to left) case assembly, case inside view, printer holder
- Case will be printed in several pieces connected with finger joints, in order for easy printing, assembly, charging, and troubleshooting.
- Printer holder will be screwed in with 3/32 screws via holes on bottom piece
- Button holes are socketed from inside for snap fit with Cherry MX keys
